Bank System

A simple Java banking system developed to practice Java exception handling.

Features

* Deposit money
* Withdraw money
* Transfer money between accounts
* Custom exception handling
* Balance validation
* Input validation using throw and throws

Custom Exceptions

* InvalidAmountException
* InsufficientBalanceException

Concepts Practiced

* Exception Handling (try-catch)
* throw
* throws
* Custom Exceptions
* Object-Oriented Programming (OOP)
* Method Reusability

Project Structure

* BankAccount.java – Banking operations
* Main.java – Test scenarios
* InvalidAmountException.java
* InsufficientBalanceException.java

Purpose

This project was created to strengthen Java exception handling concepts by implementing a simple banking system with realistic error scenarios.
